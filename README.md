# 麻雀 役・聴牌判定器

[麻雀 役・聴牌判定器 Web版](https://mandel59.github.io/mahjong/)

## Web版 使い方

- 1m〜9m: 萬子
- 1p〜9p: 筒子
- 1s〜9s: 索子
- 0m, 0p, 0s: 赤ドラの5萬, 5筒, 5索
- 1z, 2z, 3z, 4z: 東, 南, 西, 北
- 5z, 6z, 7z: 白, 發, 中

手牌の入力では略記ができます。

- 1112345678999m

手牌に14枚入力した場合、一番右の牌が和了牌扱いになります。
